# Railway Booking System

_Railway Booking System_ is built on **React JS** and **Spring Boot**.

## Technologies

### Frontend

- React
- React Query
- Zustand
- Ant Design
- TailwindCSS

### Backend

- Spring Boot (v3.3.1)
- Spring Data JPA
- Spring Validation
- Spring Security + JWT Token
- PostgreSQL
- Mapstruct
- Lombok
- Swagger (Open API)
